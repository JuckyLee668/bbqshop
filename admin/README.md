# 手工烤面筋管理后台

基于 vue3-element-admin 模板开发的管理后台系统。

## 技术栈
- Vue 3
- Element Plus
- Vue Router
- Pinia
- Axios
- TypeScript

## 安装依赖
```bash
npm install
```

## 开发
```bash
npm run dev
```

## 构建
```bash
npm run build
```

## API配置
在 `.env.development` 和 `.env.production` 中配置API地址：
```
VITE_API_BASE_URL=http://localhost:3000/v1
```
