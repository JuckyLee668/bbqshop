# TabBar 图标说明

请将以下图标文件放置在此目录下：

## 图标要求
- 尺寸：81px × 81px
- 格式：PNG
- 背景：透明

## 需要的图标文件

### 首页 (home)
- `home.png` - 未选中状态（灰色）
- `home-active.png` - 选中状态（橙色 #FF6B35）

### 分类 (category)
- `category.png` - 未选中状态（灰色）
- `category-active.png` - 选中状态（橙色 #FF6B35）

### 购物车 (cart)
- `cart.png` - 未选中状态（灰色）
- `cart-active.png` - 选中状态（橙色 #FF6B35）

### 订单 (order)
- `order.png` - 未选中状态（灰色）
- `order-active.png` - 选中状态（橙色 #FF6B35）

### 我的 (profile)
- `profile.png` - 未选中状态（灰色）
- `profile-active.png` - 选中状态（橙色 #FF6B35）

## 图标建议
- 可以使用 iconfont、iconify 等图标库
- 或者使用设计工具（如 Figma、Sketch）导出
- 确保图标清晰，适合小尺寸显示
